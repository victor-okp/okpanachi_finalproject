# okpanachi_finalproject


# Project Overview
This repository contains my final project for the **Data Wrangling (Fall 2025)** course.  
The goal of this project is to assess hoisehold perception of climate chnage impact on their water security in low-income communities of Dar es Salaam, Tanzania.
# Dataset
The data used for this project include a community based survey on the demograhphic informations of the households, their knowledge of climate change, perception of how climate change affects their water security and how they are adapting and mitigating the effects.
